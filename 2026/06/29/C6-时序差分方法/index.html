C6 时序差分方法

发表于2026-06-29|更新于2026-06-29|强化学习
|总字数:7.2k|浏览量:

temporal-difference（TD，时序差分）方法是强化学习中用于价值估计的一类广泛的算法。其与 MC 方法一样，也是 model-free 的方法，但相比 MC 方法的优势是以 **incremental（增量式）**的形式进行。TD 算法的数学本质是 stochastic approximation（随机近似）。

入门：一种估计 state values 的 TD learning 方法

方法内容

在上一节介绍 RM 算法的基础上，考虑使用 RM 算法去估计 state values。

给定一个策略 π\piπ，目的是估计所有的 vπ(s)v_{\pi}(s)vπ​(s)。假设遵循策略 π\piπ，采集到的经验样本为 (s0,r1,s1,…,st,rt+1,st+1,… )(s_0,r_1,s_1,\dots,s_t,r_{t+1},s_{t+1},\dots)(s0​,r1​,s1​,…,st​,rt+1​,st+1​,…)，其中 ttt 表示时间步，则下面的 TD 算法可以利用这些样本来估计 state values：

vt+1(st)=vt(st)−αt(st)[vt(st)−(rt+1+γvt(st+1))],vt+1(s)=vt(s),for all s≠st\begin{align} v_{t+1}(s_t)&=v_t(s_t)-\alpha_t(s_t)\left[v_t(s_t)-\left(r_{t+1}+\gamma v_t(s_{t+1})\right)\right],\\ v_{t+1}(s)&=v_t(s), \qquad\text{for all $s\neq s_t$} \end{align} vt+1​(st​)vt+1​(s)​=vt​(st​)−αt​(st​)[vt​(st​)−(rt+1​+γvt​(st+1​))],=vt​(s),for all s=st​​​

其中 t=0,1,2,…t=0,1,2,\dotst=0,1,2,…，vt(s)v_t(s)vt​(s) 是状态价值 vπ(s)v_{\pi}(s)vπ​(s) 在 ttt 时刻的估计，αt∈(0,1)\alpha_t\in(0,1)αt​∈(0,1) 是 ttt 时刻对于状态 sts_tst​ 的学习率。

上面的更新公式实际上是在说，在时刻 ttt 时，只更新此刻采集到的状态 sts_tst​ 的价值，而其它状态价值全部保持不变。

注意该 TD learning 方法只用于估计给定策略 π\piπ 下的 state values，不涉及策略改进（并且也无法用于策略改进，因为 model-free 无法估计出 action values）。

方法推导

上面的 TD 算法是由 RM 算法推导过来的。

给定策略 π\piπ，对于任意状态 sss，根据状态价值的定义，有

vπ(s)=E[Gt∣St=s]=E[Rt+1+γGt+1∣St=s]=E[Rt+1+γvπ(St+1)∣St=s]\begin{align} v_{\pi}(s)&=\mathbb{E}\left[G_{t}\mid S_t=s\right]\\ &=\mathbb{E}\left[R_{t+1}+\gamma G_{t+1}\mid S_t=s\right]\\ &=\mathbb{E}\left[R_{t+1}+\gamma v_{\pi}(S_{t+1})\mid S_t=s\right] \end{align} vπ​(s)​=E[Gt​∣St​=s]=E[Rt+1​+γGt+1​∣St​=s]=E[Rt+1​+γvπ​(St+1​)∣St​=s]​​

最后一个等式实际上是贝尔曼公式的另一种形式，被称为 Bellman expectation equation（贝尔曼期望公式）。

对于 ttt 时刻的状态 sts_tst​，定义关于其状态价值的函数 ggg：

g(v(st))≐v(st)−E[Rt+1+γvπ(St+1)∣St=st]g(v(s_t))\doteq v(s_t)-\mathbb{E}\left[R_{t+1}+\gamma v_{\pi}(S_{t+1})\mid S_t=s_t\right] g(v(st​))≐v(st​)−E[Rt+1​+γvπ​(St+1​)∣St​=st​]

对于输入 vt(st)v_{t}(s_t)vt​(st​)，我们能够得到 g(vt(st))g(v_t(s_t))g(vt​(st​)) 的观测值 g~\tilde{g}g~​：

g~(vt(st))=vt(st)−(rt+1+γvπ(st+1))=(vt(st)−E[Rt+1+γvπ(St+1)∣St=st])⏟g(vt(st))+(E[Rt+1+γvπ(St+1)∣St=st]−[rt+1+γvπ(st+1)])⏟η\begin{align} \tilde{g}(v_t(s_t))&=v_{t}(s_t)-(r_{t+1}+\gamma v_{\pi}(s_{t+1}))\\ &=\underbrace{\left(v_t(s_t)-\mathbb{E}\left[R_{t+1}+\gamma v_{\pi}(S_{t+1})\mid S_t=s_t\right]\right)}_{g(v_t(s_t))}+\underbrace{\left(\mathbb{E}\left[R_{t+1}+\gamma v_{\pi}(S_{t+1})\mid S_t=s_t\right]-\left[r_{t+1}+\gamma v_{\pi}(s_{t+1})\right]\right)}_{\eta} \end{align} g~​(vt​(st​))​=vt​(st​)−(rt+1​+γvπ​(st+1​))=g(vt​(st​))(vt​(st​)−E[Rt+1​+γvπ​(St+1​)∣St​=st​])​​+η(E[Rt+1​+γvπ​(St+1​)∣St​=st​]−[rt+1​+γvπ​(st+1​)])​​​​

因此根据 RM 算法，ttt 时刻 sts_tst​ 的更新公式为

vt+1(st)=vt(st)−αt(st)g~(vt(st))=vt(st)−αt(st)[vt(st)−(rt+1+γvπ(st+1))]\begin{align} v_{t+1}(s_t)&=v_{t}(s_t)-\alpha_t(s_t)\tilde{g}(v_t(s_t))\\ &=v_t(s_t)-\alpha_t(s_t)\left[v_t(s_t)-\left(r_{t+1}+\gamma v_{\pi}(s_{t+1})\right)\right] \end{align} vt+1​(st​)​=vt​(st​)−αt​(st​)g~​(vt​(st​))=vt​(st​)−αt​(st​)[vt​(st​)−(rt+1​+γvπ​(st+1​))]​​

其中 αt(s),s∈S\alpha_t(s),s\in\mathcal{S}αt​(s),s∈S 是满足 RM 定理条件的正系数。上一部分（本章开头）提到的估计 state values 的 TD 算法与上述公式与不同之处在于，右边式中 vπ(st+1)v_{\pi}(s_{t+1})vπ​(st+1​) 被替换为了 vt(st+1)v_t(s_{t+1})vt​(st+1​)。这实际上是合理的，因为我们没法获得精准的状态价值 vπ(st+1)v_{\pi}(s_{t+1})vπ​(st+1​)（我们的目的就是为了获得 vπv_{\pi}vπ​），所以使用 ttt 时刻状态 st+1s_{t+1}st+1​ 的估计值去替代。并且，这种替代后的算法也是收敛的，这可以由拓展的 Dvoretzky’s 收敛定理证明。

Sarsa：一种估计 action values 的 TD learning 方法

上一部分的 TD 算法只能估计 state values，无法直接用于策略改进（因为 model-free 无法从 state values 推导出 action values）。这一部分介绍的 TD 算法 Sarsa 可以直接估计 action values，并且与策略改进结合可以找到最优策略。

方法内容

给定一个策略 π\piπ，目标是估计所有的 qπ(s,a)q_{\pi}(s,a)qπ​(s,a)。假设遵循策略 π\piπ，采集到的经验样本为

(s0,a0,r1,s1,a1,…,st,at,rt+1,st+1,at+1,… )(s_0,a_0,r_1,s_1,a_1,\dots,s_t,a_t,r_{t+1},s_{t+1},a_{t+1},\dots) (s0​,a0​,r1​,s1​,a1​,…,st​,at​,rt+1​,st+1​,at+1​,…)

则下面的 Sarsa 算法可以利用这些样本来估计 action values：

qt+1(st,at)=qt(st,at)−αt(st,at)[qt(st,at)−(rt+1+γqt(st+1,at+1))],qt+1(s,a)=qt(s,a),for all (s,a)≠(st,at)\begin{align} q_{t+1}(s_t,a_t)&=q_t(s_t,a_t)-\alpha_t(s_t,a_t)\left[q_t(s_t,a_t)-\left(r_{t+1}+\gamma q_t(s_{t+1},a_{t+1})\right)\right],\\ q_{t+1}(s,a)&=q_t(s,a), \qquad\text{for all }(s,a)\neq(s_t,a_t) \end{align} qt+1​(st​,at​)qt+1​(s,a)​=qt​(st​,at​)−αt​(st​,at​)[qt​(st​,at​)−(rt+1​+γqt​(st+1​,at+1​))],=qt​(s,a),for all (s,a)=(st​,at​)​​

其中 qt(st,at)q_t(s_t,a_t)qt​(st​,at​) 是 qπ(st,at)q_{\pi}(s_t,a_t)qπ​(st​,at​) 在 ttt 时刻的估计，αt(st,at)∈(0,1)\alpha_t(s_t,a_t)\in(0,1)αt​(st​,at​)∈(0,1) 是对于 (st,at)(s_t,a_t)(st​,at​) 的学习率。

之所以叫"Sarsa"，因为每次迭代需要用到五元组 State、Action、Reward、State、Action，即 (st,at,rt+1,st+1,at+1)(s_t,a_t,r_{t+1},s_{t+1},a_{t+1})(st​,at​,rt+1​,st+1​,at+1​)。

将上一部分估计 state values 的 TD 算法中的 vvv 替换为 qqq，将状态 sts_tst​ 替换为状态-动作对 (st,at)(s_t,a_t)(st​,at​)，将 st+1s_{t+1}st+1​ 替换为 (st+1,at+1)(s_{t+1},a_{t+1})(st+1​,at+1​)，即得到 Sarsa。

方法推导

与 TD 算法类似，Sarsa 可以看作是对 action values 形式的贝尔曼期望方程应用 RM 算法得到的结果。

给定策略 π\piπ，动作价值的贝尔曼期望方程为

基于 Sarsa 的最优策略学习

Sarsa 本身只做策略评估（policy evaluation）。要找到最优策略，需要将 Sarsa 与 policy improvement 相结合，所得到的迭代算法通常也被称为 Sarsa。

例子

这里对 nnn 的选取涉及偏差-方差的权衡：

  • nnn 较小（如 Sarsa）：TD target 中大量使用了估计值 qtq_tqt​，因此偏差较大但方差较低（因为涉及的随机变量少）
  • nnn 较大（接近 MC）：TD target 中更多地使用实际的奖励样本，因此偏差较小但方差较高（因为涉及的随机变量多）

这是强化学习中经典的 bias-variance trade-off。在实践中，nnn 的最优值取决于具体问题，通常需要通过实验来调节。

Q-learning：一种直接估计最优策略的 TD learning 方法

前文介绍的 TD 算法（包括 Sarsa 和 nnn-step Sarsa）都只能估计给定策略的价值，必须与 policy improvement 结合才能找到最优策略。而 Q-learning 可以直接估计最优动作价值，从而直接得到最优策略，这使其成为强化学习中最经典、最重要的算法之一。

方法内容

On-policy 和 Off-policy

理解 Q-learning 之前，首先需要理清两个重要概念：行为策略（Behavior Policy）和目标策略（Target Policy）。

  • behavior policy：用于生成经验样本的策略。
  • target policy：不断更新、最终收敛到最优策略的策略。

当 behavior policy 和 target policy 一样时，这样的学习过程称为 on-policy；反之若不同，则称为 off-policy。

学习方式 定义 特点
On-policy 行为策略 = 目标策略 用正在学习的策略生成样本，必须平衡探索与利用
Off-policy 行为策略 ≠ 目标策略 可以用其他策略（如人类专家、随机策略）生成的样本来学习

Sarsa 是 On-policy 算法

Sarsa 每一步需要 (st,at,rt+1,st+1,at+1)(s_t,a_t,r_{t+1},s_{t+1},a_{t+1})(st​,at​,rt+1​,st+1​,at+1​)：

st→πbat→environmentrt+1,st+1→πbat+1s_t \xrightarrow{\pi_b} a_t \xrightarrow{\text{environment}} r_{t+1},s_{t+1} \xrightarrow{\pi_b} a_{t+1} st​πb​​at​environment​rt+1​,st+1​πb​​at+1​

其中 πb\pi_bπb​ 既是生成样本的 behavior policy，也是 Sarsa 正在评估和改进的 target policy。两者相同，因此 Sarsa 是 on-policy。

Sarsa 使用 ε\varepsilonε-greedy 策略而非纯贪心策略，正是因为该策略需同时承担行为策略的角色，必须具有探索性。

Q-learning 是 Off-policy 算法

Q-learning 每一步只需要 (st,at,rt+1,st+1)(s_t,a_t,r_{t+1},s_{t+1})(st​,at​,rt+1​,st+1​)：

st→πbat→environmentrt+1,st+1s_t \xrightarrow{\pi_b} a_t \xrightarrow{\text{environment}} r_{t+1},s_{t+1} st​πb​​at​environment​rt+1​,st+1​

关键在于behavior policy πb\pi_bπb​ 负责生成 ata_tat​，可以是任意探索性策略；而 target policy πT\pi_TπT​ 是 arg⁡max⁡aq(s,a)\arg\max_a q(s,a)argmaxa​q(s,a)（纯贪心）。生成 (rt+1,st+1)(r_{t+1},s_{t+1})(rt+1​,st+1​) 的过程由环境决定，不涉及行为策略。因此，Q-learning 的目标策略与行为策略可以不同（也可以相同），是 off-policy。

Off-policy 的优势：

  1. 可以利用其他策略（如人类专家策略、随机探索策略）生成的经验数据进行学习
  2. 行为策略可以选择具有强探索能力的策略（如均匀分布），从而更充分地探索所有状态-动作对
  3. 目标策略可以是纯贪心的，不需要为了探索而牺牲最优性

注意 Off-policy 和 **Off-line（离线学习）**是两个概念，后者是指使用预先采集的数据进行学习，而不与环境交互。Off-policy 算法可以以 online 或 offline 方式实现。

实现

Q-learning 有两种实现方式：on-policy 版本和 off-policy 版本。

On-policy 版本

此版本中，行为策略与目标策略相同，均为 ε\varepsilonε-greedy 策略。

Off-policy 版本

此版本中，行为策略 πb\pi_bπb​ 与目标策略 πT\pi_TπT​ 完全分离。

目标策略是纯贪心策略而非 ε\varepsilonε-greedy：因为目标策略不需要用于生成样本，不需要探索性。这是 off-policy 的核心优势之一。

统一视角：所有 TD 算法的共同框架

本节将所有 TD 算法（包括 MC）纳入一个统一的表达框架。

统一表达式

事实上，所有 TD 算法（用于动作价值估计）都可以表示为：

qt+1(st,at)=qt(st,at)−αt(st,at)[qt(st,at)−qˉt]q_{t+1}(s_t,a_t)=q_t(s_t,a_t)-\alpha_t(s_t,a_t)\big[q_t(s_t,a_t)-\bar{q}_t\big] qt+1​(st​,at​)=qt​(st​,at​)−αt​(st​,at​)[qt​(st​,at​)−qˉ​t​]

其中 qˉt\bar{q}_tqˉ​t​ 是 TD target。不同算法的区别仅在于 qˉt\bar{q}_tqˉ​t​ 的定义不同：

MC 可以视为统一表达式的一个特例：取 αt(st,at)=1\alpha_t(s_t,a_t)=1αt​(st​,at​)=1，则 qt+1(st,at)=qˉtq_{t+1}(s_t,a_t)=\bar{q}_tqt+1​(st​,at​)=qˉ​t​（即直接用样本回报更新）。

这些算法的根本区别在于它们求解的数学问题不同：

  • Sarsa、nnn-step Sarsa、MC：求解给定策略 π\piπ 的贝尔曼期望方程，用于策略评估
  • Q-learning：直接求解贝尔曼最优方程，直接得到最优价值

因此：

  • Sarsa 类算法是 on-policy 的（评估的是生成样本的策略）
  • Q-learning 是 off-policy 的（求解 BOE 不依赖于特定的行为策略）
